# Anki Oldies Plugin

## About the Tool
This repository contains the code for my Anki Oldies Plugin. This is a simple plugin which allows you to move specifically dated cards and leeches into given decks.

## Installation

Either place the root folder into your local Anki "addons21" folder or get it directly via AnkiWeb [tba].


## Usage
_Old Cards_: Define the time range of the cards you want to move. This is done via the text field (input a positive number and the given operator) and by choosing the target deck you wish to move the cards into.

_Leeches_: Define a deck you wish to move the leeches into.
